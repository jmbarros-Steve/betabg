extractUrl() extrae la URL del mensaje. quickScrapeUrl() luego agrega "https://" si falta protocolo. Si la URL original ya tenía "https://", el resultado es "https://https://tienda.cl". Apify retorna 400 y el scrape del prospect falla silenciosamente.

▶ Ver fix
function normalizeUrl(url: string): string {
  // Eliminar double-protocol si existe
  url = url.replace(/^https?:\/\/https?:\/\//i, 'https://');
  if (!/^https?:\/\//i.test(url)) url = 'https://' + url;
  return url.trim().replace(/\/$/, '');
}
