5. Fetch sin Timeout - Crons Colgados (HIGH)

7 archivos - AbortSignal.timeout() agregado a todos los fetch

FIXED

Todos los fetch a APIs externas (Anthropic, Gemini, internal sync) no tenian timeout. En Cloud Run con max 300s, un fetch colgado bloquea el container completo.

Impacto: Previene containers colgados indefinidamente. 15 fetch calls ahora tienen timeout explicito.
