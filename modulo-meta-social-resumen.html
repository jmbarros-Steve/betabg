⚙️ Como Funciona (flujo simplificado)

De conectar la cuenta a optimizar campanas con IA.

1. Conexion — OAuth o Leadsie

Usuario
→
MetaConnectionWizard
→
OAuth / Leadsie Webhook
→
platform_connections

El token se encripta y guarda. Para Leadsie (BM Partner), el SUAT vive en env var y los scopes se hardcodean.

2. Sincronizar Metricas

Cron cada 6h
→
sync-all-metrics
→
Meta Graph API
→
campaign_metrics / adset_metrics

Trae spend, impressions, clicks, conversions de los ultimos 30 dias por campana y adset.

3. Crear y Gestionar Campanas

CampaignCreateWizard
→
manage-meta-campaign
→
Meta Marketing API

Wizard de 5 pasos: objetivo → audiencia → presupuesto → creativos → revision. CRUD completo de campanas, adsets, ads.

4. IA: Copies + Criterio + Reglas Automaticas

generate-meta-copy
→
Claude / GPT
→
criterio-meta (493 reglas)
→
Reglas auto 9am

Steve genera copies, Criterio los puntua, y las reglas automaticas pausan/activan ads segun performance.

5. Espiar Competencia

1 campo: "Nike"
→
resolvePageId
→
Apify Scraper
→
competitor_ads + metricas

Input inteligente: nombre, @handle, o URL. El backend resuelve y trae ads con impressions, spend, reach via Apify.

6. Publicar en Instagram / Facebook

SocialPublisher
→
publish-instagram / publish-facebook
→
Graph API

Publicacion directa o programada. Calendario visual. Metricas por post (reach, engagement, saves).

📨 3 Webhooks

Endpoints que reciben datos de servicios externos.

meta-data-deletion

Meta llama cuando un usuario pide borrar sus datos (GDPR). Sin auth — requerido por Meta.

leadsie-webhook

Leadsie notifica cuando un cliente acepta la conexion BM Partner. Guarda assets en DB.

leadsie-google-webhook

Leadsie para Google Ads. Mismo patron que Meta pero para cuentas de Google.
